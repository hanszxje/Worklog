# Nhật ký công việc: Hoàn thiện Web Portal Glassmorphism, Reverse Proxy Nginx, SSL & E2E Test

> **Tuần 12 - 12/07/2026:** Báo cáo tiến độ chi tiết hàng tuần.

---

### Các hoạt động và kiến thức tích lũy trong tuần

- Lập trình giao diện Web Portal (ReactJS, Node.js) hỗ trợ đa ngôn ngữ (i18n), Light/Dark mode Glassmorphism, tích hợp bảo mật xác thực 2 lớp TOTP (MFA), phân quyền 7 vai trò người dùng (RBAC), bản đồ Mapbox GL JS hiển thị cửa hàng, biểu đồ Plotly.js thể hiện doanh số actual vs forecast và cảnh báo hụt hàng.
- Triển khai Web Application lên máy chủ EC2 Ubuntu, cấu hình PM2 chạy nền liên tục, Nginx làm Reverse Proxy và cài đặt chứng chỉ SSL HTTPS qua Certbot Let's Encrypt.
- Thực hiện kiểm thử liên hợp toàn diện (E2E Test) trên môi trường Cloud với các vai trò người dùng (Sales Staff, Store Manager, Director) để xác nhận các quyền hạn hoạt động chính xác.
- Hoàn thiện bộ tài liệu báo cáo thực tập tốt nghiệp cuối khóa, Worklog, Project Proposal và Workshop hướng dẫn triển khai bằng tiếng Anh và tiếng Việt. Nộp bài báo cáo cuối kỳ.
