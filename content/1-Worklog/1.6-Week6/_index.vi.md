# Nhật ký công việc: Đánh giá EC2 Instance Types, EBS Storage & AWS Backup

> **Tuần 6 - 31/05/2026:** Báo cáo tiến độ chi tiết hàng tuần.

---

### Các hoạt động và kiến thức tích lũy trong tuần

- Nghiên cứu các nhóm Amazon EC2 Instance Types, so sánh sự khác biệt về hiệu năng và chi phí giữa CPU Intel, AMD với dòng chip tối ưu AWS Graviton.
- Tìm hiểu cơ chế lưu trữ khối Amazon EBS (các loại gp2, gp3) cùng Instance Store.
- Nghiên cứu cơ chế co giãn tự động Auto Scaling, bộ cân bằng tải Elastic Load Balancing (ELB - ALB/NLB) và các dịch vụ lưu trữ tệp tin dùng chung Amazon EFS, Amazon FSx.
- Thực hành Lab 13 cấu hình AWS Backup lập lịch sao lưu tự động cho EC2 và RDS, chạy thử nghiệm khôi phục dữ liệu (Restore Test).
